Preset
Choose a preset of mouse interaction settings.
Hide when mouse is at border
  • Hide when mouse is at border
    Hide the controls when moving the mouse to the left or right border
  • Show when mouse is over controls
    Show the controls only when moving the mouse over the bottom controls or to any border
  • Custom
Trigger Distance
The threshold from each side at which the controls will be hidden.
All values are in pixel. Set the value to 0 pixels to disable interaction with the corresponding side. Disabled sides are colored red.
Inverts the function of the mouse trigger so that they make the controls visible instead of hiding them.
Only hide the controls when the YouTube™ video is in fullscreen mode.
Hotkey
The hotkey toggles the visibility of the video controls.
Note that mouse movement will reshow the controls even if you just pressed the hotkey. If you don't like that, consider disabling the mouse interaction.
